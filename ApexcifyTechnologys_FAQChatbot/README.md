# Task 2 FAQ Chatbot

This task implements a Frequently Asked Questions chatbot using Python and basic natural language processing techniques.

The chatbot reads questions from a dataset and returns the most relevant answer based on user input.

## Features
1. Question and answer matching
2. Text preprocessing using NLP
3. Web based interface using Flask
4. CSV based FAQ dataset

## How to Run
1. Open terminal
2. Navigate to this folder
3. Run:
   python app.py
4. Open browser and go to:
   http://127.0.0.1:5000
