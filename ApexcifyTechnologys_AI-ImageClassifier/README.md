# Task 3 AI Image Classifier

This task implements an AI based image classification system trained using Teachable Machine and executed using TensorFlow in Python.

The model classifies facial expressions into predefined categories.

## Classes
1. Happy
2. Sad
3. Neutral

## Features
1. Image classification using trained AI model
2. Confidence score for each prediction
3. Testing on multiple images
4. Clean and simple Python implementation

## How to Run
1. Open terminal
2. Navigate to this folder
3. Run:
   python AI_Image-Classifier.py
